# sherpa

## User Story

```
AS A traveler 
I WANT to see the daily weather and get suggestions for things to do locally 
SO THAT I can optimize my vacation time
```

## Acceptance Criteria

```
GIVEN I am a traveler
WHEN I input my destination
THEN I am presented with that location's 5 day weather forecast
WHEN I select an option from the dropdown
THEN the google map loads pins and a corresponding list of activies is presented
WHEN I click on the activity in the list
THEN the map moves to the corresponding pin
WHEN I revisit the page
THEN I see the destinations I've previously searched
```
